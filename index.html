Instructions

1. Type the name of the sheet in the input field.

2. Choose an excel file by clicking the button "Choose File".

3. Choose row to add options for x and y axis.(By default selected row is 0).

4. Add row and column to the table created by clicking the "Add Row" and "Add Column" button respectively.

5. Edit the table created if needed.

6. Select x and y axis to make a bar graph based on it(please select valid options).

7. Download the updated excel file.

Note: Row number starts from 0.
